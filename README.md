# Astronaut

網頁小遊戲原型，已整理成正式專案結構。

## 專案結構
- `index.html`
- `style.css`
- `game.js`
- `assets/`
  - `sfx/collect.wav`
  - `sfx/dialog.wav`
  - `sfx/gear.wav`
  - `sfx/repair.wav`
  - `sfx/board.wav`
  - `images/`（預留）

## 操作
- 方向鍵：移動 / 駕駛太空船
- Space：啟動吸塵器
- T：切換穩定器
- M：開關小地圖
- E：與機器人互動 / 對話
- F：搭乘或離開太空船
- Esc：關閉對話框

## 本版內容
- 大地圖 + 攝影機跟隨
- 視差星空
- 小地圖
- 太空垃圾吸取與清理計數
- 資源回收站與回收站機器人
- 收集 25 個垃圾後可拿到齒輪零件
- 把齒輪交給太空船旁的維修機器人後修好太空船
- 修好後可搭乘太空船在地圖移動
- 已改用 `assets/sfx/*.wav` 實體音效檔播放

## 執行方式
直接用瀏覽器打開 `index.html` 即可。
若本機瀏覽器對音效有限制，請先點擊頁面再操作，或用本機伺服器開啟。
